# Board Project

React, TypeScript, NestJS, Prisma, and SQLite를 사용해 만드는 풀스택 게시판 프로젝트입니다.

이 프로젝트의 목표는 단순한 CRUD 구현이 아니라, 프론트엔드와 백엔드가 HTTP/API 계약을 기준으로 연결되는 흐름을 직접 구현하며 이해하는 것입니다.

## Tech Stack

- Node 24
- pnpm
- React
- TypeScript
- Vite
- Tailwind CSS
- NestJS
- Prisma
- SQLite
